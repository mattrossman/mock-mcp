## Setup

Install dependencies

```bash
pnpm install
```

Start containers

```bash
pnpm supabase:start
```

## Start

Serve functions

```bash
pnpm supabase:functions
```

Serve frontend

```bash
pnpm dev
```


## Cleanup

Stop containers

```bash
pnpm supabase:stop
```
